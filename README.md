# Chilan-AI-language-Learning
AI based Language Learning Platform
Taken the circumanstance below into consideration:
You are learning a foreign language and you encounter a problem of translating a sentence meaning "What do you usually do?"
You are used to typing "What do you on usual?"
The traditional Language Learning App will show wrong as the dismatch between the correct answer and youy answer.

I use a three tier evaluation "Regex - Vector Similarity - LLM",
which can solve the problem perfectly and can also enhance your understanding as
"What do you on usual?" is also a correct answer because the vector similarity of the two answers are very high.
When you type some answer that is really wrong, the platform will introduce the LLM system to analyze you answer and give you feedback.
